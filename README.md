**Kid World** is a zero-latency, gamified educational web application designed for early childhood learning. Built with a modern, interactive UI, it features regional language support, an AI voice engine, and a cloud-powered backend.

🚀 **Play the Game Live:** [https://himanshu03021994.github.io/KidWorld-3Tier/](https://himanshu03021994.github.io/KidWorld-3Tier/)

---

## ✨ Key Features

* **🎮 Gamified Learning:** Dynamic 6-button layout with sequential learning (A to Z).
* **🗣️ AI Voice Engine:** Native Web Speech API talks to kids ("Find A", "अ कहाँ है?") with regional accents.
* **⚡ Zero-Latency:** Pre-fetches cloud data on the dashboard for instant gameplay.
* **🌍 Regional Support:** Serves localized datasets based on the user's selected State.
* **❤️ Health & Rewards:** 3-lives system and real-time Coin counter (🪙).
* **🪄 Magic Reveal:** Correct answers transform letters into visual objects (e.g., `A` ➡️ `🍎 A for Apple`).

---

## 🏗️ 3-Tier Architecture

1. **Presentation Tier:** Built with HTML/JS, hosted on GitHub Pages.
2. **Logic Tier:** RESTful API built in **Go (Golang)**, hosted on Render Cloud.
3. **Data Tier:** Structured JSON maps serving rich datasets mapped to regions.
